## 🛠️ How to Run Locally

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Variables**
   Create a `.env` file in the root directory and add:
   ```env
   VITE_GROQ_API_KEY=your_groq_api_key_here
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```
